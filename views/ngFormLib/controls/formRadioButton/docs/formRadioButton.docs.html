Form RadioButton FormRadioButton.js

ngFormLib.controls.formRadioButton

Form RadioButton is a directive that is a wrapper for a label,input type="radio" and some auxillary HTML elements, designed to work within Bootstrap form-structures.

It works this way because the label,input type="radio" and error-display elements are inter-related. For example, when the required attribute is set to true the <label> element is changed so that a required-indicator-CSS-class is applied to the element.

Plugin dependency

Form RadioButton requires the form controls service module, the field error controller module , the error message container module and the FormRadioButtonTemplate.html template to be loaded.

Examples

Use form-radio-button to create a complete HTML structure containing a label and input type="radio" element that reacts to the state of the field.

Usage

Add the form-radio-button directive to an element, and supply a label as the content of the element (see example above).

Options

Options can be passed via attributes on the directive.

Name Type Default Description
ff-ng-value expression The value to use in the model-value when the radio-button is selected.
template URL See Form Controls Service options The HTML template to use for the control. The default value can be overridden on the element, or globally by setting the default value in the Form Controls Service

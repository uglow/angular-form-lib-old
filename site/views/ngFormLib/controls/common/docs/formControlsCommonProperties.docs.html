uid string generated The control's unique identifier, which is used in the id="..." attribute on the control and to link the label to the form field element. When the form field is inside a repeater, use variable as in uid="...{{$index}}", so that the control has a page-unique id, which is required to make the control fully accessible. If the uid is not specified, one will be generated by the formControlService. name string generated The name of the element, which is used to make form-validation work correctly. Only use an {{interpolationExpression}} inside this attribute when using form-radio buttons inside ng-form inside ng-repeat (see example). If the uid is not specified, one will be generated by the formControlService.

form-radio controls that are inside a radio-button-group MUST specify this attribute explicitly for the grouping-behaviour to work!

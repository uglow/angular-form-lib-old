Form Checkbox FormCheckbox.js

ngFormLib.controls.formCheckbox

The Form Checkbox directive acts like a macro in that it expands into a <label>,<input type="checkbox"> and some auxillary HTML elements, but does change the $scope of those elements.

It works this way because the elements are inter-related. For example, when the required attribute is set to true the <label> element is changed so that a required-indicator-CSS-class is applied to the element.

Plugin dependency

FormCheckbox requires the form controls service module, the field error controller module , the error message container module and the FormCheckboxTemplate.html template to be loaded.

Examples

Use form-checkbox to create a complete HTML structure containing a label and input type="checkbox" element that reacts to the error state of the field.

Usage

Add the form-check directive to an element, and supply a label as the content of the element (see example above).

Options

Options can be passed via attributes on the directive.

Name Type Default Description
ff-ng-true-value expression The value to use in the model-value when the checkbox is checked.
ff-ng-false-value expression The value to use in the model-value when the checkbox is not checked.
template URL See Form Controls Service options The HTML template to use for the control. The default value can be overridden on the element, or globally by setting the default value in the Form Controls Service
